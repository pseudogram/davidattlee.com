What's the difference between Managed or Unmanaged Resource?

David Attlee

25 Sep 2018 • 1 min read

When programming there is the concept managed and unmanaged resources. Simply, a managed resource are things that your garbage collector can see and aid in managing memory. Thus when a program is done with it, the garbage collector knows to remove all references.

Unmanaged resources include:

  • Files
  • Open connections /ports
